My Projects

Water Consumption Tracking System

In this project, I have developed a system that allows users to input their daily or monthly water usage data, and the system will calculate and provide insights into their water consumption patterns. The backend would use MySQL to store water usage data for each user, including factors such as time, volume of water consumed, and average consumption trends. The system can also generate reports on excessive usage or suggest water-saving tips based on the data.

WAPDA Electricity Project

In my second semester, I developed a comprehensive project focused on electricity management using JAVA. The application is designed to collect user input regarding electricity consumption and subsequently provide detailed information about the units consumed. The project integrates a MySQL database in the backend to efficiently store and retrieve user data. By leveraging JAVA's robust programming capabilities and MySQL's reliable data management, the system ensures accurate and real-time tracking of electricity usage, offering valuable insights to users for better energy management.
